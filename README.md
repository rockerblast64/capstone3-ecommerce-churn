# capstone3-ecommerce-churn
Capstone Project Module 3 - Customer Churn Prediction on E-Commerce using XGBoost

Project Overview
  Proyek ini bertujuan membangun model klasifikasi untuk memprediksi pelanggan yang berisiko churn pada perusahaan e-commerce. Dengan model ini, tim bisnis dapat melakukan tindakan retensi yang lebih efisien dan tepat sasaran.

Business Objective
  Mendeteksi pelanggan yang berisiko churn sebelum mereka benar-benar berhenti.
  Mencapai recall ≥ 70% untuk pelanggan churn sebagai metrik utama.
  Memberikan insight dari fitur yang paling memengaruhi churn.

Machine Learning Approach
  Algoritma: Logistic Regression, Random Forest, XGBoost
  Final model: XGBoost
  Hasil akhir:
    Recall: 75%
    F1-score: 0.80
    AUC: 0.95

Features Used
  Tenure, Complain, SatisfactionScore, DaySinceLastOrder, PreferedOrderCat, MaritalStatus, dll.
